# City Events API

Build a Restful CRUD API for listing all events around your city.

## Steps to Setup

1. Install dependencies

```bash
npm install
```

2. Start mongodb in another tab
```bash
mongod
```

3. Run Server

```bash
node run dev
```

You can browse the apis at <http://localhost:3000>
